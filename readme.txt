Social media integration icons in the side bar 

== Description ==

The plugin will have the following posiible options

1- Wrapper class for all icons   
    2- Class for single icon   
    3- width of the box  
    4- Height of the box  
    5- Box color   
    6- Widget margin (padding)  
    7- Upload Icon png  
    8- Add the link to the icon  
    9- Title for the widget

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
